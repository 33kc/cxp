## keybinds reference

| Keybind | Function |
| :--- | :--- |
| **`q`** | Exits program. |
| **`p`** | Toggles metadata permissions column (`drwxrwxrwx`) on and off. |
| **`Arrow Up`** | Moves selection cursor up |
| **`Arrow Down`** | Moves selection cursor down |
| **`Enter`** or **`Arrow Right`** | Opens selected directory  |
| **`u`** or **`Arrow Left`** | Goes back to parent directory (`cd ..`). |
| **`c`** | Copies file - (source, destination) |
| **`d`** | Deletes file |
| **`m`** | Moves file into destination, or renames it |
